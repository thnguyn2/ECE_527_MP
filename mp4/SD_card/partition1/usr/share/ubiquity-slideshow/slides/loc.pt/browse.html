Navegar na web

O Ubuntu inclui o Mozilla Firefox para navegação rápida e segura na Internet. É fácil de usar e é apoiado por uma fundação sem fins lucrativos que adora a Internet. Se não é um fã do Firefox, há também muitas alternativas no Centro de Software Ubuntu.

Programas incluídos

  • Navegador web Firefox

Programas suportados

  • Flash

  • Chromium
