Naviguez sur le Web

Ubuntu inclut Mozilla Firefox pour une navigation sur le Web rapide et sûre. Facile à utiliser, celui-ci est soutenu par une fondation à but non lucratif qui aime le Web. Si vous n’êtes pas fan de Firefox, de nombreuses alternatives sont également disponibles dans la Logithèque Ubuntu.
