Navegue por la web

Ubuntu incluye Mozilla Firefox para una navegación en la web rápida y segura. Es sencillo de usar y está respaldado por una fundación sin ánimo de lucro que ama la web. Si no es fan de Firefox, hay muchas alternativas en el Centro de software de Ubuntu.

Software incluido

  • Navegador web Firefox

Software compatible

  • Flash

  • Chromium
