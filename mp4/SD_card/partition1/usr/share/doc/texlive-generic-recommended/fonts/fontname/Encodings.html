Encodings

The following sections define various encodings as PostScript encoding vectors. These files can be installed in the Dvips header directory and then used in psfonts.map to change font encodings of downloadable or builtin fonts (see psfonts.map).

You can also specify these .enc files to Afm2tfm when building a font (see Changing font encodings).

For a discussion of TeX font encodings in general, and defining new ones for use with fontinst in particular, see http://www.ctan.org/fonts/utilities/fontinst/doc/encspecs.

  • 8a: Adobe standard.
  • 8r: TeX base.
  • cork: Cork (aka EC, TeX Latin 1, tex256).
  • dvips: Dvips.
  • texmext: TeX math extension.
  • texmsym: TeX math symbol.
  • texmital: TeX math italic.
  • texnansi: From Y&Y.
  • texnansx: From Y&Y.
  • xl2: OT1 + ISO Latin 2 (extended).
  • xt2: Typewriter version of xl2.
