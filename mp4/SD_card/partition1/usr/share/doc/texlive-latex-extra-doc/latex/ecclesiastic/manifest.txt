This package contains the following files besides this file manifest.txt:

ecclesiastic.dtx		the Work and source code
ecclesiastic.ins		the installation file
ecclesiastic.pdf		the documentation obtained by LaTeXing the dtx file
ecclesiastic.sty		the style file extracted from the dtx file with the ins file
ecclesiastic-sample.pdf	a sample text obtained by running LaTex with this extension

Installation

Install ecclesiastic.sty in your personal/local texmf tree  rooted in <root>, in a folder named  <root>/tex/latex/ecclesiastic/
	
Install ecclesiastic.dtx and ecclesiastic.ins in <root>/source/latex/ecclesiastic/

Install ecclesiastic.pdf and ecclesiastic-sample.pdf in <root./doc/latex/ecclesiastic/

Refresh the file name database if required by your distribution of the TeX system.
